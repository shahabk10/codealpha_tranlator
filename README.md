🌐 CodeAlpha: Language Translation Tool

📋 Project Overview

This project is part of my **AI Internship at @CodeAlpha**. It is a professional-grade, web-based translation dashboard that allows users to translate text across 100+ languages instantly with a clean UI/UX.


✨ Key Features

* Real-time Processing**: Fast and reliable translation using the Google Translate engine.
* Auto-Language Detection**: Automatically identifies the source language of the input text.
* Professional Dashboard**: Enhanced with custom CSS for a modern, interactive user experience.
* Intern Metadata Sidebar**: Displays intern name, domain, and system status for a professional look.
* Copy to Clipboard**: Integrated code-block feature for quick translation copying.

🛠️ Tech Stack

* Language**: Python 3.13
* Framework**: Streamlit (Web UI)
* API**: Deep-Translator (Google API Wrapper)
